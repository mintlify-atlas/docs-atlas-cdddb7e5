## What is shopMaster?

shopMaster is a Windows desktop application that bridges your local SQL Server product database with Digible's cloud platform. Built with Python and featuring an intuitive GUI, shopMaster automatically tracks changes in your product catalog and syncs them in real-time to the cloud.

Whether you're managing a single store or multiple locations, shopMaster eliminates the manual work of keeping your online and offline product data in sync. Set it up once, and let it run in the background while you focus on your business.

## How it works

shopMaster uses a three-step approach to keep your data synchronized:

1. **Track** - Database triggers automatically log changes to a `ChangeLog` table whenever products are inserted, updated, or deleted
2. **Sync** - Every 2 minutes, shopMaster reads the ChangeLog and syncs changes to Digible's API at `api.digible.one`
3. **Clean** - After successful sync, the ChangeLog is cleared and ready for the next batch of changes

<Info>
  shopMaster requires SQL Server with ODBC Driver 17 and Python 3.x to run on your Windows machine.
</Info>
